Random Generators API

Mid Squares

Random Generated numbers

/generated/midSquares

Example

Parameters
  • seed : The initial seed for the algorithm
  • reps : The number of desired random numbers
Response
  • Seed : The seeds array
  • Generator : The square of the seed divided in 3 parts in the form xx|xxxx|xx array
  • Generated : The generated number array
  • Randoms : The generated number between 0 and 1 array

Random Generated numbers with goodness tests

/goodness/midSquares

Example

Parameters
  • seed : The initial seed for the algorithm
  • reps : The number of desired random numbers
  • acceptance : The percentage of acceptance of the goodness tests in a scale from 0 to 1
Response
  • Randoms:
    • Seed : The seeds array
    • Generator : The square of the seed divided in 3 parts in the form xx|xxxx|xx array
    • Generated : The generated number array
    • Randoms : The generated number between 0 and 1 array
  • Goodness:
    • ChiSquare
      • Chi ^2 calculated : The result of the ChiSquare Algorithm
      • Chi ^2 Theoretical : The result in the Chi Square Tables for the acceptance given
      • Accepted : True | False if the set of random numbers is under ChiSquare
    • Kolmogorov
      • Kolmogorov Distance : The result of the Kolmogorov Algorithm
      • Theory Distance : The result in the Kolmogorov Tables for the acceptance given
      • Accepted : True | False if the set of random numbers is under Kolmogorov
